## Quick Start

1. **Clone the Repository**
   ```bash
   git clone https://github.com/dibakarmitra/portfolio-nextjs.git
   cd portfolio-nextjs
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```
